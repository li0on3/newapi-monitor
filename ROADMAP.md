# 路线图

[简体中文](ROADMAP.md) | [English](ROADMAP_EN.md)

路线图按实际需求和故障数据推进，不承诺固定发布日期。

## 当前基线（1.0）

- 外置、只读集成 New API，不修改上游源码。
- 渠道真实探测、日志耗时分析、资源监控和采集器自监控。
- 企业微信、飞书与邮件多渠道告警及页面测试。
- New API Session SSO、角色映射、紧急管理员和配置审计。
- 单机 Docker Compose 快速部署、备份、诊断和安全加固。

## 下一阶段

- 发布带版本号的 GHCR 多架构镜像与自动生成的 GitHub Release。
- 增加配置导入/导出，但秘密字段默认不导出。
- 增加通知模板预览、静默时段和告警升级策略。
- 增加独立外部心跳示例，用于检测整机失联。
- 持续补充可观测性、恢复演练和升级兼容测试。

## 暂不引入

- 默认不引入 Prometheus、Grafana、Loki、消息队列或外部时序数据库。
- 默认不做多节点控制面和复杂插件系统。

只有单机容量、数据保留或可靠性目标出现明确证据时，才重新评估这些组件。
